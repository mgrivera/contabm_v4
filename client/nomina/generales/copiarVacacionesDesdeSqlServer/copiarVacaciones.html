Este proceso lee las vacaciones registradas en Contab y las copia a este programa. La idea es, a partir de la ejecución de este proceso, comenzar a registrar y editar las vacaciones solo desde este programa, y nunca más desde el anterior.

Este proceso no podrá ser ejecutado dos veces; el proceso detecta cuando las vacaciones ya existen en este programa y deja de efectuar la copia. La idea es que no exista la posibilidad de copiar las mismas vacaciones más de una vez.

Nótese que este proceso copia las vacaciones solo para la compañía que se ha seleccionado.
