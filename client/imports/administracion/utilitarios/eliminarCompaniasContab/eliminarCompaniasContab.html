Eliminar compañías Contab

Este proceso permite eliminar todos los datos asociados a la compañía Contab seleccionada, y que han sido registrados en la base de datos por los usuarios del programa.

Entre los datos que elimina este proceso están: facturas y pagos, movimientos bancarios, asientos contables, cuentas contables, registros de saldos contables, información de caja chica, etc.

Como indicamos más abajo, el objetivo de este proceso es eliminar todos los datos que corresponden a la compañía Contab seleccionada, aunque no ella misma, la cual debe ser eliminada desde el catálogo de compañías Contab.

Nota: este proceso elimina todos estos datos en forma permanente y sin posibilidad de recuperarlos.

Por este motivo, es recomendable que Ud. se asegure muy bien que la compañía Contab seleccionada corresponde a que Ud. realmente desea eliminar.

Aunque este proceso elimina datos para la compañía seleccionada, no elimina la compañía como tal. Para hacerlo, luego de la ejecución de este proceso, Ud. debe abrir el catálogo de compañías y eliminar la compañía. Nótese que ésta no debe ser la compañía seleccionada, para poder ser eliminada. Es decir, antes de eliminar la compañía, y luego de ejecutar este proceso, Ud. debe seleccionada otra, ir al catálogo de compañías y eliminarla.
