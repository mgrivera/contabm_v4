Nota: el valor indicado para Mínimo es usado para calcular y aplicar la retención de impuesto solo si la base del mismo supera este monto.

Note que Ud. tiene tres opciones para este valor: 1) deje en blanco para que el mínimo sea establecido en base a la unidad tributaria; 2) ponga cero (0) para que siempre se calcule una retención de impuesto; es decir, que nunca apique un mínimo; 3) ponga un monto para que la retención solo aplique si la base del impuesto supera el mínimo.
